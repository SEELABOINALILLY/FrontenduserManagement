Project Description

This is a User Management Web Application built using HTML, CSS, JavaScript (Frontend) and Spring Boot + MySQL (Backend API). The application allows users to view user data fetched from a live REST API deployed on Railway.

The frontend is deployed using GitHub Pages, while the backend is hosted on Railway cloud platform, making the project fully cloud-connected.

🚀 Features
Fetch and display user details from REST API
Clean and responsive UI design
Search functionality for filtering users
Real-time data integration with Spring Boot backend
Deployed frontend using GitHub Pages
Cloud-hosted backend using Railway
🔧 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Spring Boot (Java)
Database: MySQL
Deployment: GitHub Pages + Railway
🌐 Live Architecture

Frontend (GitHub Pages) → REST API → Spring Boot (Railway) → MySQL Database

🎯 Purpose

This project demonstrates full-stack development skills including REST API integration, database connectivity, and cloud deployment.
